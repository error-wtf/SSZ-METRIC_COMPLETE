# SSZ Observable Data Registry

This folder contains the structured registry of physical observables utilized to validate the canonical, pure Segmented Spacetime (SSZ) metric.

## Files

- `observable_registry.json`: The fully serialized metadata registry for all active physical and mathematical observables mapped to the Prime Directive classes and routing methods.

## Integrity Policies

- **Anti-Circularity**: No model parameters (such as segment density $\Xi(r)$ or characteristic scaling) are permitted to be fitted against the observational data in this directory.
- **Forward Prediction**: Every observable must be calculated in a direct, forward sequence starting from the analytical definitions.
