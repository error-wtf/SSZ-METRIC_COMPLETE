# Fetching NICER Data via HEASARC

The public NICER data is fetched via the HEASARC Master Catalog `nicermastr` using coordinate searches.

## HEASARC Archive Reference

- Catalog: `nicermastr`
- Download hosts: GSFC (HEASARC), AWS S3 bucket (`s3://heasarc-nicer`), SciServer
- Required reduction package: HEASoft / NICERDAS
